## Multiplayer ASCII
### terminal, bombs, and multiplayer. What more could you want.

##### Requires: Python 3 with pip and virtualenv (venv)

#### Controls
- WASD - Movement
- F - Throw bomb
- L - Lay mine

#### Launching
- Launch either unix-launcher or windows-launcher
- If this results in an error please open an issue!
